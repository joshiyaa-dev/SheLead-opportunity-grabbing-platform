💜 SheLeads — Women's Opportunity Intelligence Platform
A beautifully designed, fully functional web app that helps women discover, evaluate, and apply for jobs, internships, hackathons, and scholarships.
✨ Features

Smart Match Engine — Calculates a % match score between your skills and each opportunity
Readiness Levels — Ready (≥70%), Partially Ready (40–69%), Not Ready (<40%)
30+ Opportunities — Jobs, Internships, Hackathons, Scholarships
Skill Selector — Search from 90+ skills or add your own custom skills
Filters — Filter by type (Job/Internship/etc.) and readiness level
Apply Tracking — Mark opportunities as applied and see them in one place
Smart Suggestions — Learn which skills to build to unlock more opportunities
Botanical UI — Sage green, lavender, warm cream palette inspired by botanical boutique aesthetics

🚀 How to Deploy on GitHub Pages

Create a new GitHub repository (e.g. sheleads)
Upload index.html to the root of the repo
Go to Settings → Pages
Under Source, select main branch and / (root) folder
Click Save
Your site will be live at: https://yourusername.github.io/sheleads

🛠 Tech Stack

Pure HTML, CSS, JavaScript — zero dependencies
Google Fonts (Playfair Display + Lato)
No build step needed — just open index.html in a browser
